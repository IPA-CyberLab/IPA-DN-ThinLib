これらのヘッダファイルは、Microsoft 社の
「Microsoft Windows SDK for Windows 7 and .NET Framework 4」
C:\Program Files (x86)\Microsoft SDKs\Windows\v7.1A
からコピーしたものを修正したものです。

修正内容は、元 SDK ファイル「fwptypes.h」の #define によるマクロ定義で
空行が誤って挿入されていることから、Visual Studio 2019 の VC++ コンパイラ
でコンパイルエラーになる問題があり、この問題が Microsoft 社によって修正
されていないことから、やむを得ず当該空行を削除した修正済みファイルを
"fixed_" というファイル名プレフィックスを付加した上で、
本ソースコード中に取り込んだものです。

いずれも元ファイルは関数および定数の一覧が並べられているヘッダファイル
であり、プログラムの著作物としての性質を有するものではないと考えますが、
念のため出典とコピーを行なった理由と、修正を行なった内容をここに
記載しておくものであります。
